# 🎮 Rock Paper Scissors Game

A fun and interactive Rock Paper Scissors game built using **HTML**, **CSS**, and **JavaScript**. Challenge the computer, track your score, and enjoy a clean and responsive user interface.

🚀 Perfect for practicing JavaScript fundamentals, DOM manipulation, and game logic.

🎯 Features

- Play Rock, Paper, or Scissors against the computer
- Random computer choice generation
- Real-time game results
- Score tracking system
- Reset game functionality
- Responsive design
- Modern and clean user interface
- Interactive hover effects

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

⚙️ How to Play

1. Click on one of the choices:
   - ✊ Rock
   - ✋ Paper
   - ✌️ Scissors

2. The computer will randomly select its move.

3. The result will be displayed:
   - 🎉 You Win
   - 😢 Computer Wins
   - 🤝 Draw

4. Scores are updated automatically after each round.

5. Click the Reset Game button to start over.

🎮 Game Rules

Rock beats Scissors
Scissors beats Paper
Paper beats Rock
